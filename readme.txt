📝 极地特快 - 最终验收规格说明书
1. 核心场景与环境
地图布局：非圆形简单轨道，采用**“三叶草”或“8字形”复杂闭合曲线**，有穿梭感。
生态系统：
地标 (POI)：轨道环绕中心有 3 棵巨型地标圣诞树（高亮、有星星）。
自然森林：大量普通松树作为背景。
智能避障：所有树木、路灯、石头必须严格避开轨道（至少 15-18米净空），绝不穿模，不遮挡行车视野。
氛围感：黑夜背景、大雾（Fog）、积雪地面、月光照明（HDR 高动态范围）、Bloom 辉光特效（灯光晕染）。
2. 火车本体 (高精度 & 防脱节)
模型细节：
车头：必须有锅炉、驾驶室、前方烟囱、前方排障器、车轮连杆。
车厢：带弧形车顶和窗户，数量 6-8 节。
物理连接：
间距：车厢之间距离极近。
连接杆：必须有物理模型（连接轴）相互插入，转弯时视觉上不断开。
行驶逻辑：
车头朝向必须正确（正向行驶，非倒车）。
轮子随速度转动。
烟囱冒烟（粒子向后飘）。
3. 摄像机与交互 (TPS 驾驶模式)
上帝视角 (God Mode)：
自由轨道控制器，不可穿地。
驾驶跟随 (Follow Mode) - 核心痛点修复：
位置：默认锁定在车尾后上方，视线看向车头前方。
距离记忆：支持鼠标滚轮/双指缩放改变跟随距离，系统应记住这个距离，不强行重置。
智能回正：用户拖拽旋转看侧面时，暂停自动跟随角度；松手后，镜头平滑、柔和地自动转回车尾正后方。
交互：双击屏幕切换上述两种模式。
4. UI 与参数 (产品化体验)
启动页：全屏引导，介绍玩法，点击“开始”后镜头从高空俯冲入场。
设置面板 (GUI)：
位置：点击右上角齿轮弹出，居中显示，适配手机。
全中文参数：包含速度、跟随距离/高度、阻尼平滑度、月光/灯光强度、雾气浓度、车厢间距等 30+ 项参数。
